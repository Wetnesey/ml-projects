## Основные проекты и исследования

### Generative AI & Deep Learning
* **GAN Paper Reproduction:** Реализация архитектуры Generative Adversarial Networks с нуля.
    * *Стек:* PyTorch, CNN, Generative Models.
    * *Результат:* Успешная тренировка модели и воспроизведение визуальных результатов из оригинальной статьи.
* **CV Fundamentals:** Реализация полносвязных и сверточных (CNN) нейросетей, включая кастомные слои и backpropagation.

### Natural Language Processing (NLP)
* **Embedding-based Text Ranking:** Система ранжирования текстов на основе семантической близости векторов. Применение Metric Learning в задачах поиска.
* **Next Token Prediction:** Разработка языковых моделей для предсказания следующего токена. Работа с вероятностными распределениями в текстах.
* **RNN Text Classification:** Классификация последовательностей с использованием рекуррентных нейросетей (LSTM/GRU).

### Classical ML & Data Engineering
* **Decision Trees & Boosting:** Исследование ансамблевых методов. Тонкая настройка градиентного бустинга (XGBoost, LightGBM).
* **Robust Data Preprocessing:** Построение ETL-пайплайнов: работа с выбросами, заполнение пропусков, Feature Engineering и масштабирование данных.
